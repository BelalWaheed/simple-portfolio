# Belal Waheed — Full-Stack Developer Portfolio

A modern, responsive, and high-performance personal portfolio website built with **Tailwind CSS**, **JavaScript**, and **HTML5**. This portfolio showcases my projects, skills, and professional journey as a developer based in Cairo.

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Cursor Glow Effect**: Interactive background glow that follows the mouse movement (Desktop only).
- **Scroll Animations**: Smooth reveal-on-scroll animations for sections and skill cards.
- **Dynamic Projects Section**: Showcases curated work with direct links to GitHub repositories.
- **Modern UI/UX**: Dark-themed aesthetic with glassmorphism and subtle micro-animations.
- **Contact Integration**: Easy access to social profiles (LinkedIn, GitHub, Instagram) and email.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern utility-first styling.
- **JavaScript (Vanilla)**: Interaction and logic.
- **Phosphor Icons**: High-quality vector icons.
- **Google Fonts**: Inter & JetBrains Mono typography.

## 📂 Project Structure

- `index.html`: Main entry point and structural layout.
- `style.css`: Custom styling, utility definitions, and animations (e.g., cursor glow, blobs).
- `script.js`: Interactive logic, scroll observers, and cursor follow effects.

## 🛠️ Setup & Usage

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/BelalWaheed/simple-portfolio.git
   ```
2. **Open `index.html`**:
   Simply open the file in your preferred browser, or use a local development server like VS Code's "Live Server".
